# 🚀 M3U Editor Pro Elite

M3U Editor Pro Elite is a powerful, lightweight, and **privacy-first** web tool for managing IPTV lists. 

The main goal of this project is to provide a secure alternative to closed-source editors. **Your M3U links are private, and they should stay that way.**

## 🛡️ Privacy & Security (The "Reddit Test")
This tool is built to be 100% transparent:
* **100% Client-Side:** All processing happens in your browser's RAM using JavaScript.
* **No Backend:** Hosted on Blogger/GitHub Pages, which means there is no PHP or Database to store your links.
* **The Offline Test:** You can load the site, disconnect your internet, and the editor will still work perfectly. No data is ever sent to external servers.

## ✨ Key Features
* 🖱️ **Drag & Drop:** Easily reorder your channels.
* ⭐ **Favorites Manager:** Organize your top-tier content.
* 🧹 **Duplicate Remover:** Clean your list with one click.
* 🕵️ **Offline Detection:** Identify dead links (via client-side fetch).
* 📝 **Advanced Support:** Preserves `#KODIPROP` tags and DASH (`.mpd`) manifests (Coming soon).

## 🚀 How to use
Simply visit [https://m3ueditorpro.blogspot.com/](https://m3ueditorpro.blogspot.com/), upload your M3U file, edit it, and download your cleaned version.

# 🚀 M3U Editor Pro Elite (Versión en Español)

M3U Editor Pro Elite es una herramienta web potente, ligera y **enfocada en la privacidad** para la gestión de listas IPTV.

El objetivo principal de este proyecto es ofrecer una alternativa segura a los editores de código cerrado. **Tus enlaces M3U son privados, y así es como deben permanecer.**

## 🛡️ Privacidad y Seguridad (El "Test de Reddit")
Esta herramienta ha sido construida para ser 100% transparente:
* **100% Lado del Cliente (Client-Side):** Todo el procesamiento ocurre en la memoria RAM de tu navegador mediante JavaScript.
* **Sin Backend:** Alojado en Blogger/GitHub Pages, lo que significa que no existe PHP ni bases de datos para almacenar tus enlaces.
* **El Test Offline:** Puedes cargar el sitio, desconectar tu internet y el editor seguirá funcionando perfectamente. Nunca se envían datos a servidores externos.

## ✨ Funciones Principales
* 🖱️ **Arrastrar y Soltar:** Reordena tus canales fácilmente.
* ⭐ **Gestor de Favoritos:** Organiza tu contenido de primer nivel.
* 🧹 **Eliminador de Duplicados:** Limpia tu lista con un solo clic.
* 🕵️ **Detección de Caídos:** Identifica enlaces rotos (vía fetch desde el cliente).
* 📝 **Soporte Avanzado:** Preserva etiquetas `#KODIPROP` y manifiestos DASH (`.mpd`) (Próximamente).
