# 🚀 M3U Editor Pro Elite

M3U Editor Pro Elite is a powerful, lightweight, and **privacy-first** web tool for managing IPTV lists. 

The main goal of this project is to provide a secure alternative to closed-source editors. **Your M3U links are private, and they should stay that way.**

## 🛡️ Privacy & Security (The "Reddit Test")
This tool is built to be 100% transparent:
* **100% Client-Side:** All processing happens in your browser's RAM using JavaScript.
* **No Backend:** Hosted on Blogger/GitHub Pages, which means there is no PHP or Database to store your links.
* **The Offline Test:** You can load the site, disconnect your internet, and the editor will still work perfectly. No data is ever sent to external servers.

## ✨ Key Features
* 🖱️ **Drag & Drop:** Easily reorder your channels.
* ⭐ **Favorites Manager:** Organize your top-tier content.
* 🧹 **Duplicate Remover:** Clean your list with one click.
* 🕵️ **Offline Detection:** Identify dead links (via client-side fetch).
* 📝 **Advanced Support:** Preserves `#KODIPROP` tags and DASH (`.mpd`) manifests (Coming soon).

## 🚀 How to use
Simply visit [https://m3ueditorpro.blogspot.com/](https://m3ueditorpro.blogspot.com/), upload your M3U file, edit it, and download your cleaned version.
